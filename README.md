
# Narzary Engineering & Consultancy Pvt. Ltd.

Narzary Engineering & Consultancy is a multidisciplinary firm providing professional services in civil engineering, IT solutions, planning & design, and citizen support. We deliver innovative, reliable, and sustainable solutions for clients across government, private, and institutional sectors.

## Business Areas

- **Civil Engineering Works**: Constructions, infrastructures, buildings/houses, roads & bridges, estimation & costing, planning & design.
- **Consultancy Services**: DPR preparation, project management, estimation & costing, IT-related services, roads/bridges/buildings.
- **IT Services**: Software development, app development, web hosting, maintenance, ERP solutions, cloud & DevOps, GIS, cybersecurity.
- **Planning & Design**: Architectures, residential/commercial, town/city planning, interior/exterior design, space planning.
- **Citizen Services**: ITR filings, tax exemption (ST), government schemes, training, skill development, documentation support.
- **Career & Skill Development**: Job openings, internships, certifications, workshops, bootcamps, career counseling, mentorship.

## Our Approach

- Client-focused solutions tailored to project needs
- Technical excellence and regulatory compliance
- Sustainable, cost-effective, and practical designs
- Transparent communication and project management

## Technology Stack

- **Frontend**: Next.js, React, Radix UI, TypeScript
- **Backend**: Node.js, Express, REST APIs
- **Cloud & DevOps**: Azure, AWS, CI/CD pipelines
- **GIS & Data**: Geospatial analysis, mapping tools

## Getting Started

To run the project locally:

```bash
pnpm install
pnpm dev
```

To check types:

```bash
pnpm typecheck
```
